291    (*traversal-functions* :*) is called with two arguments,

320  (defmulti expand-1
321    "macro-like facility for rte.
322    A call to this function expands an rte pattern (once).
323    Methods are responsible for expanding as a function of the first element
324    of the pattern e.g., (and ...), (spec ...), (:? ...) etc.
325    Methods take two arguments [pattern functions]
326    pattern is the entire rte pattern being expanded, which might be sequence or otherwise.
327      E.g., pattern = (:and A B C)
328      or    pattern = :empty-set
329    A method may return either a transformed version of the pattern,
330    or may return the pattern itself.   This function is called within
331    a call to fixed-point which will keep calling the function until it
332    eventually _expands_ into itself.
333    The :default method returns the given pattern, so as fixed-point
334    continues to expand the pattern, eventually there'll be no other
335    applicable method and the :default method will be called, triggering
336    fixed-point to terminate."
337    (fn [pattern _functions]
338      (cond
339        (not (sequential? pattern))
340        :default
341        :else
342        (first pattern))))
