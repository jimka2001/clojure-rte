  \begin{columns}
    \begin{column}{0.55\textwidth}
  \begin{itemize}
  \item Mathematical notation:\\
  \quad\textcolor{greeny}{$(Int \cdot (Float \cup Ratio)^+)^*$}

    \only<1>{\usebox\exnoteboxpre}%
    \only<2>{\usebox\exnotebox}%
    \begin{itemize}
    \item Either: Leaf node designating a Clojure/Java type
    \item Or: List of keyword followed by more RTEs
    \end{itemize}
  \item Semantics: an  RTE corresponds to a \Emph{set of sequences}: Union, Intersection
